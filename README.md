# Mythril Dashboard

Optional web dashboard for managing your Mythril instance. Built with React + Vite.

## Features

- View brain contents
- Monitor bot status
- Configure settings

## Running

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`
